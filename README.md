## Mr Price-Web Development
Mr Price website is a responsive website built with HTML, CSS and JavaScript.
It is designed mobile-first with responsive layout, typography and navigation.
The Mr price mission and vision it is to provide fashion valu and quality products to our customers at an affordable prices.
The website's goal is to ensure that we supply good and long lasting products to our customers.
##Features
The website has a HTML structure for accessibility and clarity, it is custom designed: it has chosen colour palette. 
This website has a responsive layout that works well on mobile, tablet and desktop. It contains a separate CSS and Javascript files for an organised code
## Tech Used 
HTML for page structure and content
CSS for styling, layout, colours, fonts and responsiveness.
Javascript for the interactive behaviour and user actions
